# Guida Uso - Copertec v3.3

## Avvio

1. Apri copertec-configuratore-v3.3.html
2. Compila dati cliente
3. Aggiungi lucernari
4. Calcola BOM
5. Esporta

## Sezioni

### Dati Cliente
- Ragione Sociale (obbligatorio)
- Referente, Email, Telefono
- Note

### Lucernari
- Nome identificativo
- Schema (A, A1, A2, B, B1, B2, C, C1, C2, D, D2, E, E2)
- Quantita
- Dimensioni (larghezza cm, lunghezza m)
- Altezza rete (152, 183, 213 cm)

### Ottimizzazione
- Standard: rotoli dedicati
- Equilibrata: riutilizza >= 4m
- Max Risparmio: riutilizza >= 5m

### Sconti
- Reti (%)
- Accessori interni (%)
- Accessori esterni (%)
- Accessori giunzione (%)

### Trasporto
- Franco fabbrica
- Franco destino

## Export

- PDF: preventivo professionale
- Excel: BOM completa
- WhatsApp: messaggio riepilogo
- Email: draft con dati

## Note

- Schemi giuntati (A2, B2, C2, D2, E2) = x2 rotoli
- Verificare sempre riepilogo sfridi
